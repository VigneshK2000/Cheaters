In order to run the program,
0. Type make to create executable
1. Type the executable "./plagiarismCatcher"
2. Then, type in the path to the directory of your desired text files to check
3. Type in the number of words to chunk by
4. Type in the threshold for a suspecting plagiarism
5. Enter

The names of the files are main.cpp, HashTable.cpp, and HashTable.h
